# 📊 E-commerce Sales Data Analysis

A complete data analysis project that explores sales data to uncover key business insights using Python, Pandas, and Matplotlib.

---

## 📌 Project Overview

This project performs end-to-end data analysis on an e-commerce sales dataset.  
It includes:

- Data loading and cleaning  
- Exploratory data analysis  
- Sales performance evaluation  
- Data visualization using charts  
- Insight generation for business decisions  

---

## 🎯 Objectives

- Analyze total and average sales  
- Identify top-performing products  
- Understand regional sales distribution  
- Study sales trends over time  
- Present results using visualizations  

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- Matplotlib  
- VS Code
